# GCP Serverless Labs

Google Cloud serverless learning labs covering Cloud Run, Cloud Functions, App Engine, and related deployment patterns.

## Purpose

This repository is a course/reference workspace for practicing GCP serverless services. It is useful as a lab archive, not as a single deployable application.

## Contents

- Cloud Run examples.
- Cloud Functions examples.
- App Engine examples.
- Dataproc/PySpark and support scripts.
- Notes and artifacts from hands-on GCP practice.

## How To Use

Navigate into a specific lab folder and follow that lab's deployment files or scripts. Treat each folder as an independent exercise.

## Status

Active reference repo. The main improvement is to add per-lab READMEs with prerequisites, commands, expected cloud resources, and cleanup steps.

## Validation

Each lab should define the expected cloud resource, the command that deploys it, the command that verifies it, and the command that removes it.

## Engineering Notes

Serverless labs become useful long-term only when prerequisites, IAM assumptions, regions, and cleanup steps are documented.
